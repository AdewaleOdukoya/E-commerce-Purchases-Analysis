# E-commerce-Purchases-Analysis

## Overview
This project explores an e-commerce purchases dataset to understand customer behaviour, payment methods, and purchasing patterns.

## Objectives
- Analyse purchase price distribution
- Identify most common email providers
- Examine payment methods
- Explore purchase timing patterns

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Key Insights
- Most purchases occur during PM hours
- Gmail is the most common email provider
- A large proportion of customers use MasterCard
- Engineers represent a significant customer group

## Dataset
E-commerce purchase dataset containing:
- customer emails
- purchase price
- payment provider
- credit card expiration
- job titles
